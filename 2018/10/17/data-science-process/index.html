数据分析流程与相关技术

Posted on 2018-10-17 | Edited on 2018-10-18 | Views:

数据科学中一项任务的流程通常由以下5步组成：

  1. 获取数据
  2. 数据预处理
  3. 数据分析
  4. 分析报告
  5. 化洞见为行动

下面简要介绍整个过程以及其中涉及到的主要技术或工具。

获取数据(Acquire Data)

获取数据的过程主要涉及以下任务：

  • 寻找数据集
  • 提取数据
  • 查询数据

数据的来源非常之多，访问的方式各不相同，下面列举几个常见的来源：

  • 传统的数据库 - 如 MySQL, Oracle等，通常使用 SQL 进行查询。
  • 文本文件 - 如 CSV, JSON 等，可使用脚本语言处理。
  • 远程数据 - 通过 Web 服务进行访问。
  • NoSQL 存储 - 如 Redis, MongoDB, HBase 等，通过其 API 或 Web 服务访问。

准备数据(Prepare Data)

获取数据之后，首先需要探索数据以形成对现有数据的理解，随后对数据进行必要的预处理，为下一阶段的数据分析做准备。

探索数据(Explore Data)

探索数据的主要目的是更好的理解现有数据的特性，比如了解数据的相关性、整体趋势、离群值等。主要有以下手段：

  • 初步分析
  • 可视化

描述性统计

通过初步分析获得一些描述性的统计值，如：

  • 均值(Mean)
  • 中位数(Median)
  • 众数(Mode)
  • 范围(Range)
  • …

数据可视化

通过数据可视化更好地理解数据，如：

  • 热力图(Heat maps)
  • 直方图(Histogram)
  • 箱线图(Boxplots)
  • 线形图(Line graphs)
  • 散点图(Scatter plots)
  • …

预处理(Pre-process)

现实世界的数据往往存在各种各样的问题，例如：

  • 不一致的数据值
  • 重复的记录
  • 数据值缺失
  • 无效的数据
  • 离群值

我们首先需要定位和处理数据质量问题，比如：

  • 补充缺失值或移除存在缺失值的记录
  • 合并重复的记录
  • 对无效值生成最佳的估计值
  • 移除离群点

随后根据任务需要对数据进行必要的变换，例如：

  • 数据缩放(如归一化)
  • 数据格式转换
  • 特征选择(删除、合并、添加特征等)
  • 降维(Dimensionality Reduction)

数据分析(Analyze Data)

数据分析主要有以下几个阶段：

  • 选择合适的分析技术
  • 构建模型
  • 评估结果

数据分析技术主要分为以下几类：

  • 分类(Classification): 主要用于预测分类(离散值)。
  • 回归(Regression): 主要用于预测数值(连续值)。
  • 聚类(Clustering): 用于将数据集划分为不同的簇，同簇的数据相似度较高。
  • 关联分析(Association Analysis): 发现规则来捕获要素之间的关联关系
  • 网络分析(Graph Analytics): 使用图结构来查找实体间的连通关系。

分析报告(Reporting Insights)

为了更好地展现从数据分析得到的结论，我们往往需要添加图表来表达数据分析的结果和数值，以下是一些常用的可视化工具：

  • R 和 Python 都有一些强力的库来支持数据分析和可视化任务。
  • D3 是一个用于网页作图、生成互动图形的 JavaScript 库。
  • Leaflet 是一个轻量级且兼容移动端的 JavaScript 库，可用于创建可交互的图形。
  • Tableau Public 是一款免费的数据可视化分析工具。可用于创建和分享可视化图形。
  • Google Charts 可适配不同的浏览器并兼容不同的移动端平台。
  • Timeline 是一个可用于创建时间线的 JavaScript 库。

化洞见为行动(Turning Insights into Action)

只有将通过数据分析获得的有价值的信息转化为行动，数据分析才能真正实现其价值。后续行动需要谨慎定义和评估。

  • 依据数据分析结果来执行后续动作
  • 确定执行流程并自动化
  • 评估效益(测量，监控，评估)
  • 据此决定下一步动作
